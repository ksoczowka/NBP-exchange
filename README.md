# NBP-exchange
Simple app to get conversion from any currency to PLN from any day using requests to get information from NBP (Narodowy Bank Polski)

## Install
Windows:
```powershell
git clone https://github.com/ksoczowka/NBP-exchange.git
```
```powershell
cd NBP-exchange\
```
```powershell
.\install.bat
```

Linux:
```shell
git clone https://github.com/ksoczowka/NBP-exchange.git
```
```shell
cd NBP-exchange/
```
```shell
./install.sh
```
## Use
```
cd src
```
```
python main.py
```
